# 开发环境说明
1、安装5.6.0以上Node 
2、安装mongodb，不要配置密码
3、安装sass
4、全局安装bower
5、全局安装supervisor
6、全局安装node-inspector
7、全局安装puer
8、运行前执行命令
```
npm install
bower install
```


# node实时调试命令
```
node-inspector --web-port=8888
node --debug app.js
```

# 运行
`node app.js`，访问`http://localhost`
